# Chinmaya Panda – Developer Portfolio

A modern, responsive developer portfolio built using **React, Tailwind CSS, and Framer Motion** to showcase backend development skills, projects, certifications, and professional growth.

This portfolio reflects my focus on building secure, scalable, and production-ready backend systems using the MERN stack.
